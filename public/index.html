## ¿Qué es Visual Studio Code?

Visual Studio Code es un editor de código gratuito creado por Microsoft. Es uno de los editores más populares del mundo entre programadores y desarrolladores web.

## ¿Por qué me gusta?

- Es gratuito y de código abierto

- Tiene miles de extensiones para todo tipo de lenguajes

- Es muy rápido y ligero

- Tiene terminal integrada y control de versiones con Git

## ¿Qué es Linux?

Linux es un sistema operativo de código abierto creado por Linus Torvalds en 1991. A diferencia de Windows, Linux es gratuito y cualquier persona puede ver y modificar su código.

## ¿Por qué usar Linux?

- Es gratuito y de código abierto

- Es muy seguro y estable

- Consume menos recursos que Windows

- Es muy usado en servidores y programación

## Distribuciones populares

Las distribuciones más conocidas son Ubuntu, Debian, Fedora y Arch Linux. Cada una tiene sus características pero todas usan el mismo núcleo de Linux.
